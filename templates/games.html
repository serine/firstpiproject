Lets have some fun!

Well done, you have made a terrific effort by taking up this challenge! I am going to show some python games and of cource we are going to launch Minecraft Pi edition shortly.

Once you have logged in you should see command-line prompt like this pi@raspberrypi ~ $. Go ahead and change into python_games/ directory by typing cd python_games/ and press enter, your promopt will now look like this pi@raspberrypi ~ /python_games $ this means you are in the python_games/ directory.

To see the contents of python_games/ directory you can type ls and press enter. There will be alot of different files and you might get a bit worried since you wouldn't understand it all. Don't worry! All you need to know at this stage, any files that end with .py extension, will be your games of interest. For example to start Tetromino, which is a python version of Tetris type in python tetromino.py and press enter and should see the game appearing on the screen. To leave any game just press Esc key.

Quick introduction to Python programing langugae

You probably don't think so now by programming can be so much fun. You can make all sorts of different applications with programming. There are alot of different programming languages, just like there are many different Linux distributions, each programming language is just as valid and powerfull. Python one of the most intuitive programming languages. Python comes pre-installed with most of Linux disctributions and sure enough your Raspbian has it. Go ahead and start you python by typing python and pressing enter. You will notice that you usual prompt will change to a different prompt like this one >>>. Well done, you are at the python promp, to exit from python press ctrl^D.

Here is a few quick commands for you python

Don't forget quatation marks for any words you are going to type in python and always press enter at the end of any commands, just like you would when you typed in Linux command

  • >>> print "Hello, yourname!"
  • >>> 2+2
  • >>> word = "Hello, yourname!"
  • >>> word*5

And finally Minecraft Pi edition

We were launching games straight from the CLI. As we have learned before you can also use GUI to do exactly the same. To start GUI from CLI type in startx and press enter, you should see GUI coming up on your screen. From now on it is piece of cake, just navigate with your mouse to menu -> games -> python game OR Minecraft Pi edition.
