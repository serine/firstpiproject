Basic Linux commands for CLI

Here you are going to learn a few basic Linux commands. The great thing about Linux commands, they are independent of a distribution type. Once you have learned Linux commands, you can transfere those knowledge to any other Linux computer.

There are alot of different Linux commands, you don't need to know them all, in fact not many people would know all of Linux commands. However as your expirence with Linux CLI gets bigger so will your knowledge of different Linux commands.

Here is some useful tips on CLI and commands execution:

  • Always follow this general syntax: command space argument Enter
    e.g ls Documents/ This will list contents of Documents/ directory (folder)
  • Use "Tab Completion" feature. You don't always need to explicitly type out everything, just start typing first couple of letters of a command or an argument and then press "Tab" key and the rest of the command or an argumennt will be autocomplete for you.
  • To kill any commands or other miss behaviour by CLI use ctrl^C (that is press ctrl togehter with C key)
  • Some commands may not need any arguments e.g date, cd or ls commands if you just type ls and press enter it will list content of the current directory
