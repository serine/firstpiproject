Introduction to Command-Line Interface

Command-line interface is a user interface that is used for interacting with a computer, simply put an interface is way through which you as a user can interact with a computer. The most common user interface is graphical, that is Graphical User Interface or GUI for short. GUI is what you see when you log in to your computer at home. You see colorful windows and icons hence graphical interface. One other type of interface is Command-Line Interface or CLI for short. This interface isn't colorfull and there are no icons. However it is just as valid in interacting with a computer, infact it is much more powerfull way of interacting with a computer.
