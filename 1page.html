In order to operate your raspberry pi you need to know Linux. What is Linux? Linux is an operating system, just like Windows. To be perfectly correct Linux actually is the kernel and every modern operating systems would have a kernel as a heart of its system. (You can read more about different types of operating systems here link to Wikipedia). Despite Linux that important remake about Linux is kernel, it is very very common to call any operating systems that uses Linux kernel, simply Linux operating system or Linux for short. However there are many different Linux based operating systems they are called distributions. Probably by far most common once are Ubuntu, Linux Mint, Debian, Fedora and openSUSE, but there are really hundreds of different Linux distributions. And by the way not every Linux distribution has to be a fully functioning operating system, those mentioned above are dedicatedly full operating systems alike Windows 7 or 8 OR alike Macintosh OS X. Linux distributions can be just a firmware for a modern or a router. We are going to use Raspbian Linux distribution, which is optimised Debian distribution for raspberry pi hardware.
